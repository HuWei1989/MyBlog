2021-01-18
FFmpeg的代码
git clone https://git.ffmpeg.org/ffmpeg.git ffmpeg
X264
git clone https://code.videolan.org/videolan/x264.git
X265
git clone https://github.com/videolan/x265.git
1.安装X264

不能简单的运行sudo apt-get install nasm,通过官方下载安装https://www.nasm.us/pub/nasm/releasebuilds/2.14.01/nasm-2.14.01.tar.gz， 

        tar xvf  nasm-2.13.03.tar.gz
        ./configure
        make -j4
        make install

2.安装X265 

        cmake CMakeLists.txt
        make -j4
        make install
